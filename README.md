<h3 align="left">🌌 AR-based AI Celestial Tracking System</h3>

<p align="left">
An intelligent <b>AR-powered sky mapping system</b> designed for real-time celestial tracking and immersive astronomical visualization.  
It uses <b>geolocation, computer vision, and interactive 3D rendering</b> to accurately map <b>stars, planets, and constellations</b> in the user’s surrounding sky.  
Built to blend astronomy with advanced engineering, the system delivers a futuristic stargazing experience through <b>Python, OpenCV, PyTorch, Unity 3D, and ARKit</b>.
</p>

###

<h3 align="left">🚀 Key Features</h3>

<ul align="left">
  <li>📍 Real-time celestial object tracking using live geolocation.</li>
  <li>🧠 AI-assisted recognition and sky alignment with computer vision.</li>
  <li>✨ Interactive AR overlay for planets, stars, and constellations.</li>
  <li>🪐 3D sky mapping experience powered by Unity rendering.</li>
  <li>📷 Accurate positioning and object detection using OpenCV + PyTorch.</li>
  <li>📱 Mobile AR support with ARKit for immersive visualization.</li>
