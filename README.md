### 🌌 AR-based AI Celestial Tracking & Sky Mapping

An advanced augmented reality system for **real-time celestial tracking** using **geolocation, AI, and computer vision**.  
It enables accurate visualization of **stars, planets, and constellations** through an interactive **3D sky-mapping interface**.  
The system is built with **Python, OpenCV, PyTorch, Unity 3D, ARKit, and C#** for immersive and intelligent astronomical exploration.

### 🚀 Highlights

- Real-time sky mapping based on device location and orientation  
- AI-assisted celestial recognition and positional alignment  
- Interactive AR overlays for stars, constellations, and planets  
- Computer vision integration for scene understanding  
- 3D rendering pipeline for immersive astronomy visualization
